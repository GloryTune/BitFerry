# BitFerry 🚢

**跨平台局域网文件互传 + 即时聊天工具**，基于 PyQt6 开发，支持 macOS 与 Windows。

只需连接同一 Wi-Fi 或插入同一根网线，即可在设备之间快速传输文件、截图、文字，无需互联网、无需账号。

---

## 功能特性

- **自动发现设备** — 同一局域网内的设备自动出现在列表中，在线/离线状态实时显示
- **文字聊天** — 支持换行（Shift+Enter），消息按会话和日期分组显示
- **文件传输** — 单个文件、多个文件、整个文件夹一次性打包发送
- **截图发送** — 快捷键唤出区域截图工具，截图直接进入输入框发送（macOS 原生 / Windows 系统截图）
- **图片粘贴/拖入** — 剪贴板图片、图片文件直接拖入输入框，自动嵌入显示
- **聊天记录持久化** — 历史消息本地保存，可随时翻阅与特定设备的全部聊天记录
- **未读消息提示** — 角标、系统通知、Dock 闪烁三重提醒
- **自定义接收目录** — 收到的文件按发送方机器名自动归档，存放位置可自由更改
- **设备管理** — 右键可删除长期离线的历史设备
- **手动刷新** — 点击「↻ 刷新」按钮立即重新扫描局域网设备
- **图标内嵌** — 应用图标直接编译进程序，跨机器运行无需附带任何资源文件

## 环境要求

| 项目 | 要求 |
|------|------|
| Python | 3.8 或更高版本 |
| PyQt6 | `pip install PyQt6` |
| mss（可选）| Windows 截图备用方案：`pip install mss` |
| 操作系统 | macOS 10.14+ / Windows 10+ |

> **注意：** 防火墙需放行 UDP **50808** 端口（设备发现）和 TCP **50809** 端口（文件/消息传输）。

## 快速运行（源码方式）

### 1. 克隆仓库

```bash
git clone https://github.com/GloryTune/BitFerry.git
cd BitFerry
```

### 2. 安装依赖

```bash
pip install PyQt6
# Windows 用户（可选，用于截图功能）
pip install mss
```

### 3. 启动应用

```bash
python bitferry.py
```

## 打包为独立应用（无需 Python 环境）

### macOS — 打包为 `.app`

**前提：** 安装 PyInstaller

```bash
pip install pyinstaller
```

**执行打包：**

```bash
pyinstaller bitferry.spec --clean --noconfirm
```

打包完成后，可运行的应用位于：

```
dist/BitFerry.app
```

将 `BitFerry.app` 拖入 `/Applications` 即可像普通 Mac 应用一样使用，双击启动，无需安装 Python。

> **体积说明：** 打包后约 65–70 MB，包含完整 Qt 框架，这是 PyQt6 应用的正常体积。

---

### Windows — 打包为 `.exe`

修改 `bitferry.spec`，在 `EXE` 块中将 `console=False` 保持不变，然后在 Windows 机器上执行：

```bash
pip install pyinstaller
pyinstaller bitferry.spec --clean --noconfirm
```

输出位于 `dist\BitFerry\BitFerry.exe`（连同 `dist\BitFerry\` 文件夹一起分发）。

若需单文件版本，也可直接运行：

```bash
pyinstaller --onefile --windowed --name BitFerry bitferry.py
```

## 使用教程

### 启动与设备发现

1. 在两台（或更多）设备上同时启动 BitFerry。
2. 确保设备连接在**同一 Wi-Fi** 或接入**同一台交换机/路由器**的网线。
3. 左侧设备栏会在几秒内自动显示发现的设备，**绿点** 表示在线，**灰点** 表示曾经连接过但当前离线。
4. 若未发现设备，点击「**↻ 刷新**」按钮手动触发一次广播扫描。

---

### 发送文字消息

1. 在左侧点击目标设备。
2. 在底部输入框中输入文字。
3. 按 **Enter** 发送，**Shift+Enter** 换行。

---

### 发送文件

**方式一：拖放**
- 直接将文件/文件夹从 Finder（或资源管理器）拖入输入框。

**方式二：按钮**
- 点击工具栏「**📎 文件**」选择单个或多个文件。
- 点击「**📁 文件夹**」选择整个文件夹（内部文件结构会完整保留）。

**方式三：粘贴路径**
- 在 Finder 中复制文件（⌘C），在输入框中粘贴（⌘V），自动识别并添加为附件。

文件以「芯片」形式显示在输入框下方，点击 **✕** 可移除，准备好后点击**发送**。

收到的文件保存在对方设备的接收目录，按**发送方机器名**自动归档到子文件夹：

```
~/BitFerry_Received/
  └── MacBook-Pro/
        ├── 项目方案.pdf
        └── 设计稿.fig
```

---

### 发送截图

**方法一：工具栏按钮**
- 点击「**✂ 截图**」按钮，窗口自动隐藏，拖选区域后松手，截图进入输入框。

**方法二：全局快捷键**
- macOS 默认快捷键：`⌃⇧A`（Control+Shift+A）
- Windows 默认快捷键：`Ctrl+Shift+A`
- 可点击右上角「**⌨ 快捷键**」自定义快捷键。

截图以缩略图形式内嵌在输入框中，支持与文字混排后一起发送。

---

### 发送剪贴板图片

- 截图后（系统截图工具或 QQ 截图等），在输入框内按 **⌘V / Ctrl+V** 粘贴，图片自动进入输入框。
- 或点击工具栏「**📋 粘贴**」按钮。

---

### 查看聊天记录

- 点击聊天窗口右上角「**🕘 聊天记录**」，查看与该设备的完整历史消息（含日期分隔）。
- 聊天记录本地持久化保存，重启应用后仍可查阅。

> **注意：** 聊天窗口本身仅显示**本次启动后**的消息，历史记录须点击按钮查看。

---

### 自定义接收目录

- 左侧「**接收目录**」卡片下点击「**更改位置**」，选择新路径。
- 点击「**打开**」可直接在 Finder/资源管理器中查看收到的文件。
- 更改只影响新收到的文件，不移动已存在的文件。

---

### 管理设备列表

- **删除设备：** 右键点击左侧设备列表中的离线设备，选择「**删除设备**」，该设备及其聊天记录将被清除。在线设备无法删除。
- **手动刷新：** 点击设备列表右上方「**↻ 刷新**」按钮，立即发送广播重新扫描设备。

## 数据存储位置

| 类型 | macOS | Windows |
|------|-------|---------|
| 聊天记录 | `~/Library/Application Support/BitFerry/` | `%APPDATA%\BitFerry\` |
| 接收文件 | `~/BitFerry_Received/` | `%USERPROFILE%\BitFerry_Received\` |

---

## 网络端口说明

| 端口 | 协议 | 用途 |
|------|------|------|
| 50808 | UDP | 设备发现（广播） |
| 50809 | TCP | 文件/消息传输 |

防火墙或安全软件拦截以上端口会导致无法发现设备或传输失败，请确保两端均已放行。
